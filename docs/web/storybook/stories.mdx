## Component Story Format

Component Story Format (CSF) is an ES6 module-based standard for writing stories in Storybook by using JavaScript objects and modules.

## Create stories (UI)

To create a new story, you can click the `â•` button in the sidebar to search for your component.

To add new story from an existing story, you can use the [Controls addon](https://storybook.js.org/docs/essentials/controls) to adjust the value of a control and then save those changes as a new story.


## Edit a story

Using the Controls addon, update a control's value for a story. You can then save the changes to the story and the story file's code will be updated for you.
