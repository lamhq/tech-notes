# Deno

## Version

This document is for Deno v1.41.2.


## Deno Runtime

Deno (or Deno Runtime) is an open-source runtime for TypeScript and JavaScript.

Built on V8, Rust, and [Tokio](https://tokio.rs/).

Features:
- [secure](https://docs.deno.com/runtime/manual/runtime/permission_apis) by default
- built-in dev tools
- powerful platform APIs
- native support for TypeScript and JSX.


## Deno Deploy

Serverless platform for deploying JavaScript code to a global edge network.

Supports Deno APIs and Node.js / npm modules.
